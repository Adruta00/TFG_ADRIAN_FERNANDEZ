#   pip install -r requirements.txt
#
# NOTAS DE COMPATIBILIDAD:
#   - numpy / pandas / matplotlib: se usa >= con mínimo seguro para ambos proyectos.

#   - python-telegram-bot: se pínea a <20 porque la v20+ introduce una API

#   - ccxt y python-binance coexisten sin conflicto: ccxt es el acceso genérico
#     multi-exchange del TFG; python-binance es la librería específica del [B].
#   - requests y websocket-client son dependencias transitivas de python-binance
#     y ccxt, pero se declaran explícitamente para fijar versión mínima segura.
